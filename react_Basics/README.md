# React Basics Playground — Learning Project (TypeScript + Vite)

This project is a **learning playground** created for exploring and practicing **React with TypeScript**.

It provides a clean environment where you can experiment with different React features, test ideas, and understand how modern React applications are structured.

---

## Purpose of the Project

The goal of this project is to offer a practical and beginner-friendly setup for:

- Learning **React fundamentals**
- Practicing **TypeScript** inside React components
- Understanding common patterns such as **state**, **props**, **events**, **effects**, **routing**, **context**, and more

This project is **not intended for production use** — it is purely meant for **learning**, **testing**, and **experimentation**.
